✊✋✌️ Rock Paper Scissors Game

A simple and interactive command-line game built using Python where the user plays against the computer.

🎮 Overview

This project simulates the classic Rock, Paper, Scissors game. The user competes with the computer over multiple rounds, and the program keeps track of scores and declares the final winner.

🚀 Features

✔️ User vs Computer gameplay
✔️ Random computer choice using Python’s random module
✔️ Flexible input handling (accepts full words or single letters)
✔️ Score tracking after each round
✔️ Final result announcement (Win / Lose / Tie)

🛠️ Tech Stack
Language: Python
Concepts Used:
Loops (while loop)
Conditional statements
Dictionary mapping
Random module
⚙️ How to Run
# Run the Python file
python rock_paper_scissors.py
🧠 Game Logic
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
Same inputs → Tie
